# Nuivo Shared Content

This repository is a static content library for Nuivo, intended for deployment on **Cloudflare Pages** under a Nuivo subdomain.

## Structure

- `/index.html` — root listing page for available content
- `/assets/` — shared static assets for future content
- `/roadmaps/nac-mvp-v3/` — NAC MVP Roadmap v3 page (`index.html`) and scoped assets
- `/demos/` — placeholder for future demos

## Deployment

Deploy this repository directly to Cloudflare Pages with no build command and no framework.
